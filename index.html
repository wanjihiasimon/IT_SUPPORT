IT Support - Troubleshooting Guide
⚡ Quick Fix Cheat Sheet (Top 10 Solutions)
  • Restart the computer (fixes 70% of issues).
  • Check cables & power connections.
  • Ensure Wi-Fi/Ethernet is properly connected.
  • Run ipconfig /release + ipconfig /renew for network issues.
  • Clear temporary files (%temp%).
  • Run Windows Update & restart.
  • Check Task Manager (Ctrl + Shift + Esc) → End unresponsive tasks.
  • Check printer power & connections before re-adding printer.
  • Ensure Caps Lock is off when typing passwords.
  • Update drivers (mouse, network, display) in Device Manager.
🌐 Network Connectivity Issues
  • Check physical connection: LAN cable, Wi-Fi signal strength.
  • Restart router/switch if multiple users affected.
  • Disable/enable network adapter from Control Panel → Network Settings.
  • Useful Commands:
    • ping google.com → Test internet connectivity.
    • tracert google.com → Check path to destination host.
    • ipconfig /all → Display all network details.
    • ipconfig /release + ipconfig /renew → Reset IP address.
    • ipconfig /flushdns → Clear DNS cache.
    • nslookup example.com → Test DNS resolution.
    • netstat -an → Check active connections.
⚡ Computer / Power Issues
  • Ensure power cable is firmly connected.
  • Test wall socket with another device.
  • Remove laptop battery (if removable) and reconnect.
  • Check UPS/extension cable functionality.
🖱️ Mouse / Pointer Issues
  • Check mouse connection (USB/Bluetooth).
  • Try a different USB port or replace batteries (wireless mouse).
  • Enable touchpad via Fn + F7/F9 depending on model.
  • Update mouse drivers via Device Manager.
  • If cursor is frozen → Restart PC with Ctrl + Alt + Del → Restart.
🐢 Slow PC Performance
  • Restart the PC to clear memory usage.
  • Check Task Manager (Ctrl + Shift + Esc) → End high CPU/RAM tasks.
  • Clear temporary files → Run cleanmgr or %temp%.
  • Disable startup apps → Run msconfig → Startup.
  • Check for malware with Windows Defender or antivirus.
  • Ensure Windows Updates are installed.
  • Defragment disk (HDD only) → Run dfrgui.
  • If still slow → Consider RAM upgrade or SSD installation.
🔒 Blocking / Controlling USB Devices
  • Disable USB storage via Device Manager → Universal Serial Bus Controllers → Disable device.
  • Use Group Policy (gpedit.msc):
    • Computer Config → Admin Templates → System → Removable Storage Access
    • Set "Deny all access" for removable storage.
  • Registry method (advanced): HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\USBSTOR → Set "Start" value to 4 (disable).
  • For enterprise: Use endpoint security software (e.g., Symantec, BitLocker policies).
💻 Useful Windows (Win + R) Commands
  • cmd → Command Prompt.
  • services.msc → Windows services.
  • eventvwr → Event Viewer logs.
  • msconfig → System Configuration.
  • devmgmt.msc → Device Manager.
  • ncpa.cpl → Network connections.
  • taskmgr → Task Manager.
  • control → Control Panel.
  • appwiz.cpl → Uninstall programs.
  • diskmgmt.msc → Disk Management.
📧 Email Problems
  • Check internet connection.
  • Confirm email/password are correct.
  • Clear cache if using webmail.
  • Check mailbox quota (full mailbox causes failure).
  • Outlook: Run outlook.exe /safe.
🔑 Login / Password Issues
  • Ensure Caps Lock is off.
  • Try resetting password via self-service portal.
  • If locked out, call IT helpdesk.
  • Domain users: Ensure PC is connected to company network.
🖨️ Printer Not Working
  • Check printer is powered on & cables connected.
  • Ensure correct printer is set as default.
  • Clear print queue (control printers).
  • Restart printer service (services.msc → Print Spooler).
  • Test with another computer.
