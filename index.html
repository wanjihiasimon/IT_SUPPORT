Simon IT Support - Troubleshooting Guide
⚡ Quick Fix Cheat Sheet (Top 10 Solutions)
  • Disable Fast Startup: Control Panel → Power Options → Choose what the power button does → Uncheck "Turn on fast startup"
  • Restart the computer (fixes 70% of issues).
  • Password reset: Hold Shift + Restart → Command Prompt → net user username newpassword
  • Check cables & power connections.
  • Ensure Wi-Fi/Ethernet is properly connected.
  • ipconfig /release + ipconfig /renew for network issues.
  • Clear temporary files → %temp%
  • Run Windows Update & restart.
  • Task Manager → End unresponsive tasks (Ctrl+Shift+Esc).
  • Check printer power & re-add printer if necessary.
  • Update drivers in Device Manager.
  • Use Revo Uninstaller for stubborn apps.
  • Use MiniTool to manage partitions.
🌐 Network Connectivity Issues
  • Check LAN cable/Wi-Fi signal.
  • Restart router/switch if multiple users are affected.
  • Disable/enable adapter: Control Panel → Network Settings.
  • Commands:
    • ping google.com → Test connectivity.
    • tracert google.com → Trace route.
    • netsh int ip reset → Reset TCP/IP stack.
    • ipconfig /flushdns → Clear DNS cache.
    • netstat -an → Active connections.
⚡ Computer / Power Issues
  • Ensure power cable is firmly connected.
  • Test socket with another device.
  • Check UPS/extension cable functionality.
  • Remove & reseat battery (if removable).
🖱️ Mouse / Pointer Issues
  • Check USB/Bluetooth connection.
  • If cursor missing: Run main.cpl → Pointers → Reset scheme.
  • Try a different port or replace batteries.
  • Enable touchpad (Fn+F7/F9).
  • Update mouse drivers in Device Manager.
  • If frozen → Restart (Ctrl+Alt+Del → Restart).
🐢 Slow PC Performance
  • Restart to clear memory usage.
  • Task Manager → End high usage tasks.
  • Run cleanmgr, %temp%, and prefetch.
  • Disable startup apps (msconfig).
  • Run antivirus scan.
  • Install updates.
  • Defrag HDD (dfrgui).
  • Upgrade RAM / use SSD for speed.
🔒 Blocking / Controlling USB Devices
  • Disable USB storage via Device Manager.
  • Use Group Policy: gpedit.msc → Admin Templates → System → Removable Storage Access.
  • Registry: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\USBSTOR → Set Start=4.
  • Enterprise: Endpoint Security / DLP tools.
🛠️ Advanced IT Troubleshooting Cases
  • Blue Screen of Death (BSOD): Run whoami /scannow, check drivers in Device Manager, check dump logs in C:\Windows\Minidump.
  • Windows Boot Issues: Boot from recovery → bootrec /fixmbr, bootrec /fixboot, bootrec /rebuildbcd.
  • Disk Errors: Run chkdsk /f /r on affected drive.
  • Application Crashes: Check Event Viewer (eventvwr.msc) under Application logs.
  • Remote Desktop Issues: Confirm RDP enabled (sysdm.cpl), allow port 3389 in firewall, check NLA settings.
  • Antivirus/Security: Check Windows Security Center → Virus & threat protection, update signatures, scan logs.
📧 Email Problems
  • Check internet connection.
  • Verify credentials.
  • Clear browser cache.
  • Check mailbox quota.
  • Outlook: Run outlook.exe /safe.
🔑 Login / Password Issues
  • Ensure Caps Lock is off.
  • Use self-service password reset.
  • If locked out → Contact IT.
  • Domain: Ensure PC is on company network.
🖨️ Printer Not Working
  • Check printer power/cables.
  • Ensure default printer is correct.
  • Clear print queue (control printers).
  • Restart Print Spooler service.
  • Test with another PC.
