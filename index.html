üê¢ Slow PC Performance
  • Restart the PC to clear memory usage.
  • Check Task Manager (Ctrl + Shift + Esc) ‚Üí End high CPU/RAM tasks.
  • Clear temporary files ‚Üí Run cleanmgr or %temp%.
  • Clear temporary files ‚Üí Runprefetch.
  • Disable startup apps ‚Üí Run msconfig ‚Üí Startup.
  • Check for malware with Windows Defender or antivirus.
  • Ensure Windows Updates are installed.
  • Defragment disk (HDD only) ‚Üí Run dfrgui.
  • If still slow ‚Üí Consider RAM upgrade or SSD installation.
üîí Blocking / Controlling USB Devices
  • Disable USB storage via Device Manager ‚Üí Universal Serial Bus Controllers ‚Üí Disable device.
  • Use Group Policy (gpedit.msc):
    • Computer Config ‚Üí Admin Templates ‚Üí System ‚Üí Removable Storage Access
    • Set "Deny all access" for removable storage.
  • Registry method (advanced): HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\USBSTOR ‚Üí Set "Start" value to 4 (disable).
  • For enterprise: Use endpoint security software (e.g., Symantec, BitLocker policies).
